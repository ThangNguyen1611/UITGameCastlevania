# Castlevania
 Đồ án Nhập môn Phát triển Game - Game *Castlevania NES*
 
## Giảng viên hướng dẫn
 Thầy **Đinh Nguyễn Anh Dũng**

## Một vài kỹ thuật đã được áp dụng
- Sprite, Animation
- AABB Collision
- Swept AABB Collision
- Scene Management
- Camera
- Tilemap
- Phân hoạch không gian bằng lưới (Grid)
- Simple Game AI
